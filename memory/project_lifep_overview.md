LifeP is a personal life-planning web app built around a knowledge graph + AI feedback loop.

**Completed phases:**
- Phase 0: Foundation (Next.js 16 + TS + Tailwind + shadcn scaffold)
- Phase 1: Storage layer (Drizzle + SQLite schema, seed data, typed graph queries)
- Phase 2: Chat plumbing (Gemini 2.0 Flash via AI SDK, streaming chat UI)
- Phase 3: Graph view (React Flow canvas, custom node renderers, filter bar, split layout)

**Current phase:** Ready to start Phase 4 — AI writes to the graph (tool calling, proposals queue)

**Why:** Building toward a self-updating knowledge graph where AI auto-writes annotations and proposes structural changes for user review.

**How to apply:** Read PLAN.md before starting any new phase. Phases must be completed in order.
